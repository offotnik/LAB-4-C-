## Задача 1

Составить программу, которая проверяет, есть ли в списке L хотя бы два одинаковых элемента.

Для решения данной задачи, напишем метод HasDublicates, который принимает List. Проходимся по списку и смотрим, есть ли хотя бы один дубликат. Если есть, то выводим true, в противном случаи false.
